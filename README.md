# tinyTicket - an alternative way for low-price events

This project aims to help NGOs and associations that need a solution to sell low-price tickets for events that they host.
It is effective up to CAD$ 15.00 per ticket when using [Stripe](https://stripe.com/). You may choose another payment processor and change the code as well.
